# Проект: Место

## О проекте

Реализация интерфейса страницы пользователя социальной сети, в которой можно:
- делиться фотографиями;
- загружать новые фотографии;
- обновлять описание профиля;
- ставить лайки.

## Технологии

При разработке использовались:
- HTML;
- CSS;
- WebPack;
- JS.
